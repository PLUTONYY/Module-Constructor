# Module-Constructor
##  Назначение системы

Узел полезной нагрузки представляет собой модульное универсальное устройство, позволяющее решать задачи по:

- Обнаружению грузов переноски и дефектов ЛЭП;
- Переключению камер для удобства управления;
- Переноске грузов разных типов;
- Очистке ЛЭП;
- Индикации состояния.

---

## Инструкция для сборки модуля

1. Распечатайте все `.plgx` файлы на 3D-принтере.  
2. Соберите конструкцию в соответствии со схемой сборки, используя стяжки и термоклей.  
3. Проверьте конструкцию модулей на наличие жёсткой фиксации всех компонентов.  
4. Соберите электрическую часть модулей в соответствии со схемой электрических соединений.  
5. Проверьте электрическую схему на наличие короткого замыкания и правильность соединения проводов.

---

## Инструкция для развертывания системы

1. Подсоедините полётный контроллер к компьютеру.  
2. Откройте **Betaflight App** и подключитесь к дрону.  
3. Зайдите на вкладку **«Предустановки»** и выберите **«Загрузить резервную копию»**.  
4. Выберите файл `Constructor_bf.txt` и загрузите его в полётный контроллер.  
5. Подключите аппаратуру управления к компьютеру.  
6. Выберите режим **«USB Storage (SD)»**, после чего появится подключение на ПК.  
7. Откройте папку **`MODELS`** и переместите в неё файл `Constructor_rx.txt`.
